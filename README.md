# Kaidablo Duo — GitHub Pages Ready

## What this is
A single-page site for Pablo + Kaida with:
- home / about / shows / workshops / gallery / media / contact / links
- gallery + slideshow
- reels section
- downloads section
- localStorage-backed admin content

## How to publish on GitHub Pages
1. Create a new repository.
2. Upload everything from this folder to the repository root.
3. Make sure the main file is named `index.html`.
4. In GitHub, go to **Settings → Pages**.
5. Set the source to your main branch and root folder.
6. Save and wait for the Pages URL to activate.

## Important note
This version stores admin edits in the browser (`localStorage`).  
That means edits stay after refresh on the same device/browser, but they do not sync between devices unless you add a backend or CMS later.
