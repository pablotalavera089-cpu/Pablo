# Pablo Talavera — Artist Portfolio

Professional portfolio website for Pablo Talavera, multidisciplinary performance artist.

**Live site:** https://pablotalavera089-cpu.github.io/Portfolio/

## 🎭 Features

✅ **Pages:**
- Home / Portfolio
- Workshops
- Performances  
- Gallery
- **Testimonials Carousel** (auto-rotating + manual controls)
- **Booking Form** (saves to localStorage)
- **Media Kit** (downloadable content packs)
- Linktree (social links)

✅ **Design:**
- Minimalist, professional aesthetic
- Smooth animations & transitions
- Custom cursor tracking
- Fully responsive (mobile, tablet, desktop)
- Dark theme with gold accents

✅ **Functionality:**
- Admin authentication
- Booking form submissions saved locally
- Carousel testimonials (auto-advance every 6s)
- Integrated social media links
- Media kit download prompts
